Created2019-03-01|Updated2019-03-01

selenium（浏览器自动化测试框架）是一个用于Web应用程序测试的工具。Selenium测试直接运行在浏览器中，就像真正的用户在操作一样。

在python中也有selenium模块

pip install selenium

使用流程:

  • 导报:from selenium import webdriver
  • 实例化某一款浏览器对象
  • 自指定自动化操作代码
